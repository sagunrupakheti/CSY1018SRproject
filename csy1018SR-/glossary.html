Sass


Standing for Syntactically Awesome Style Sheets, Sass is a style sheet language.It is interpreted into CSS ( Cascading Style Sheets).It is compatible with all versions of CSS. Ruby must me pre installed to use Sass. Sass supports variables, nesting, patials, import, mixins, extend and operators.


Node Js


Node.js is an open source server environment that is free.It runs on various platforms i.e. Windows, Linux, Unix, Mac OS X, etc. It uses JS on the server. It's main feature is it uses asynchronous programming. It can generate dynamic page content with all features to use on the server. The node js files have .js extensions.


React.js


React.js is a javascript framework which was created and is used by Facebook.It is a User Interface (UI) library. It is a tool for building UI components.


Laravel Framework


Laravel is a web application framework.The aim of Laravel is to make web development functional without having to compromise.It handles the authentication part i.e.authentication,catching,sessions and routing.


Wordpress


Wordpress is the most popular open source CMS (Content Management System). It can be installed, updated and deployed for free.It covers the largest CMS market in the world. With Wordpress no designing skills are required to form a functional website.
