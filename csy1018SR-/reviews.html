JavaScript Frameworks

JavaScript Frameworks are JavaScript code libraries.They are called frameworks because they provide users with a backbone i.e. framework along which the code can be written and websites can be built.They simplify HTML coding.
There are various JS Frameworks in the market among which we tried out jQuery .

Pros :

  • Light weight compared to other JS
  • Open source software making learning easier
  • Similar to CSS so, easier to learn
  • Very little knowledge required to make good animations.

Cons:

  • Open source making solving of problems difficult
  • Some versions are less compatible
  • Being light weight can create coding problems in the future
  • Slower than CSS

The Verdict

Even though there are cons in the software, the pros seem to definitely cover them up. JQuery seems very convinient for beginner learners in the programming field as well as professionals as because of it's conviniency. As a whole, JavaScript Frameworks have made JS learning easier.
